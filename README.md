# cathaybk-test

**專案概述**

利用 Spring Boot 3 和 Java 11/17 實作，並使用 Docker 建立資料庫。實作的功能包含：

    從 API 讀取商品與價格資料並存入資料庫。

    提供查詢、新增、修改、刪除價格的 API。

    實作開始與結束時間的價格漲跌和漲跌幅計算功能。


**環境需求**

    Java: 17

    Spring Boot: 3

    資料庫: MySQL（使用 Docker 建立）


**使用技術**

    Java 語言

    Spring Boot 框架

    TomCat 伺服器

    MySQL 關聯式資料庫

    Spring Data JPA

    MVC 三層式架構

    RESTful 設計

    Maven 專案
